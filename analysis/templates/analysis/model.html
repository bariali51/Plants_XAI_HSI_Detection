Explainable AI (XAI)

Grad-CAM (Gradient-weighted Class Activation Mapping)

Grad-CAM produces visual explanations by computing the gradient of the predicted class score with respect to feature maps of the last convolutional layer. This reveals which spatial regions of the input image most influence the model's classification decision, providing transparency and interpretability for plant pathology diagnosis.

The heatmap overlay uses a JET colormap where red/warm regions indicate high model attention (likely infection zones) and blue/cool regions indicate low relevance.

Disease Progression Estimation

After Grad-CAM generation, the system analyzes the heatmap using HSV color space segmentation to estimate the infection ratio. Red/yellow regions represent infected tissue, while green regions indicate healthy tissue. The ratio of infected to total leaf area provides the disease severity percentage.
